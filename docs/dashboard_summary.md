Dashboard Summary

The Power BI dashboard presents the final gold reporting tables from the NYC 311 pipeline.

Executive Overview

The executive overview page shows total requests, closed requests, open requests, median agency close time, closure rate, requests by agency, median close time by agency, requests by borough, and data quality summary.

Complaint Analysis

The complaint analysis page shows median close time, complaint type count, total requests, largest complaint volume, slowest complaint types by median close time, and top complaint types by request volume.

Current Dashboard Assets

The repository includes screenshots for executive overview and complaint analysis.

Dashboard Scope

The current dashboard is based on a 50,000 row December 2019 sample. It demonstrates operational reporting, complaint analysis, agency performance, borough volume, and data quality visibility.

Future Dashboard Expansion

Future pages can include borough operations, agency performance detail, and a dedicated data quality page after a wider date range is loaded.
